# Datasets

## Propósito

Esta sección documenta los conjuntos de datos generados durante el desarrollo
de modelos físicos, modelos híbridos y gemelos digitales de sistemas
electromecánicos.

---

# Dataset 1

## Generador síncrono de imanes permanentes de flujo axial (AF-PMSG)

## Descripción

Dataset sintético generado mediante simulación computacional de un generador
trifásico de flujo axial.

El dataset tiene como objetivo servir como base para:

- Análisis estadístico.
- Modelos híbridos físico-datos.
- Desarrollo de sensores virtuales.
- Validación de algoritmos de diagnóstico.

---

# Generación del dataset

## Fuente

Modelo físico del generador axial.

Métodos utilizados:

- Modelado matemático.
- Simulación computacional.
- Parámetros eléctricos y electromecánicos.

---

# Variables consideradas

Ejemplos:

## Variables eléctricas

- Corrientes de fase.
- Voltajes.
- Potencia.
- Frecuencia.

## Variables mecánicas

- Velocidad.
- Torque.
- Posición.

## Variables térmicas

- Temperatura del rotor.
- Temperatura estimada.
- Variables relacionadas con pérdidas.

---

# Características

Tipo:

Datos sintéticos informados por física.

Sistema:

AF-PMSG trifásico.

Uso:

Investigación en:

- Gemelos digitales.
- Modelos híbridos.
- Detección de anomalías.
- Estimación de estados.
