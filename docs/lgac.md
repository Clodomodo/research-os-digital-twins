# Línea de Generación y Aplicación del Conocimiento (LGAC)

## Título

Modelado, simulación y validación de sistemas eléctricos y electromecánicos mediante gemelos digitales, modelos físicos y computación reproducible.

---

## Descripción

Esta línea de investigación se enfoca en el desarrollo de metodologías para el análisis de sistemas electromecánicos complejos mediante:

- Modelado acausal.
- Simulación multifísica.
- Gemelos digitales basados en física.
- Generación de datos sintéticos.
- Integración de datos experimentales.
- Sensores virtuales.
- Computación distribuida heterogénea.

---

## Sistema principal de estudio

Generadores síncronos de imanes permanentes de flujo axial (Axial Flux Permanent Magnet Generators).

Áreas consideradas:

- Dominio electromagnético.
- Dominio térmico.
- Dominio mecánico.
- Análisis de señales.
- Diagnóstico de fallas.

---

## Pregunta central de investigación

¿Cómo desarrollar gemelos digitales reproducibles y de alta fidelidad para sistemas electromecánicos utilizando modelos físicos, datos sintéticos informados por la física y arquitecturas computacionales accesibles?

## Filosofía de trabajo

La investigación busca demostrar que es posible desarrollar herramientas científicas reproducibles utilizando:

- Software abierto.
- Hardware accesible.
- Documentación transparente.
- Validación experimental.
- Control de versiones.
