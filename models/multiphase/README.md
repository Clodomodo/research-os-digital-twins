# Modelos multifásicos

## Propósito

Documentar la extensión de modelos de máquinas eléctricas desde sistemas
trifásicos hacia arquitecturas multifásicas.

La investigación considera como base los generadores de flujo axial de
imanes permanentes (AF-PMSG).

---

# Motivación

Las máquinas multifásicas ofrecen ventajas como:

- Mayor tolerancia a fallas.
- Reducción de esfuerzos eléctricos por fase.
- Aplicaciones en sistemas críticos.

El aumento del número de fases también incrementa la complejidad del modelo
matemático y computacional.

---

# Modelo base

## Sistema trifásico

Representación inicial:

Fases:

- A
- B
- C

Utilizado para:

- Generación del dataset sintético inicial.
- Validación de modelos.
- Desarrollo de sensores virtuales.

---

# Extensión a cinco fases

Objetivo:

Generalizar la metodología hacia un sistema de cinco fases.

Aspectos a estudiar:

- Transformación de variables.
- Nuevas componentes armónicas.
- Cambio en observabilidad.
- Complejidad computacional.

---

# Extensión a siete fases

Objetivo:

Evaluar la escalabilidad del modelo hacia sistemas con mayor número de
variables eléctricas.

Aspectos:

- Incremento dimensional.
- Nuevas dinámicas acopladas.
- Impacto en simulación distribuida.
